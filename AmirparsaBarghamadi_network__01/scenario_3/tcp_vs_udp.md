# tcp vs udp

## 1) تفاوت های اصلی tcp و udp

- tcp اتصال گرا است و قبل از ارسال داده اتصال برقرار می کند
- udp بدون اتصال است و بدون برقرار کردن اتصال داده را ارسال می کند

- tcp قابل اعتماد است و در صورت از دست رفتن بسته ها دوباره ارسال می کند
- udp قابل اعتماد نیست و ارسال دوباره به صورت پیش فرض ندارد

- tcp ترتیب بسته ها را حفظ می کند و داده را مرتب تحویل می دهد
- udp تضمین ترتیب ندارد و ممکن است داده نامرتب برسد

- tcp کنترل جریان و کنترل ازدحام دارد و سرعت ارسال را مدیریت می کند
- udp کنترل ازدحام و کنترل جریان داخلی ندارد

- tcp سربار بیشتری دارد و معمولا کندتر و با تاخیر بیشتر است
- udp سربار کمتر دارد و معمولا سریع تر و با تاخیر کمتر است



## 2) 

- video streaming
  udp مناسب تر است چون تاخیر کم مهم تر از از دست نرفتن کامل چند بسته است و پخش باید روان بماند

- file transfer ftp
  tcp مناسب است چون صحت کامل داده مهم است و نباید چیزی گم یا خراب شود

- dns query
  udp مناسب تر است چون درخواست ها کوتاه هستند و سرعت مهم است و اکثر پرس و جوها با udp انجام می شوند

- web browsing http https
  tcp مناسب است چون اتصال پایدار و تحویل درست داده لازم است و مرورگر باید محتوا را درست و کامل بگیرد

- voip call
  udp مناسب تر است چون تاخیر کم حیاتی است و چند بسته از دست برود بهتر از مکث و قطع شدن صدا است

- database query
  tcp مناسب است چون داده باید درست و کامل برسد و ترتیب و اطمینان در پاسخ ها مهم است
