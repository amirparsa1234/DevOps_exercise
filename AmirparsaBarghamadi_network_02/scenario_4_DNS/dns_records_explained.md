# انواع رکوردهای DNS

A
- رکورد آدرس IPv4
- مثال: example.com به 1.2.3.4 اشاره می‌کند

AAAA
- رکورد آدرس IPv6

CNAME
- نام مستعار برای یک نام دیگر
- مثال: www.example.com به example.com اشاره می‌کند

MX
- رکورد سرور ایمیل برای یک دامنه

TXT
- رکورد متنی
- معمولاً برای SPF و DKIM و تأیید مالکیت استفاده می‌شود

NS
- رکورد نام‌سرورها برای دامنه

SOA
- اطلاعات اصلی زون
- شامل شماره سریال و زمان‌های refresh و retry و expire
