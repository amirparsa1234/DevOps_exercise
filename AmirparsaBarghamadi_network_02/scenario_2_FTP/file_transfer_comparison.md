# مقایسه سریع انتقال فایل

## FTP
- پورت: ۲۱
- به صورت پیش‌فرض رمزنگاری ندارد
- ساده و قدیمی است و هنوز روی بعضی سیستم‌های قدیمی دیده می‌شود
- برای اینترنت عمومی مناسب نیست چون نام کاربری، رمز عبور و داده‌ها می‌توانند قابل شنود باشند

## SFTP
- پورت: ۲۲
- رمزنگاری شده است
- انتقال فایل روی SSH انجام می‌شود
- انتخاب پیش‌فرض خوبی برای انتقال فایل امن است

## SCP
- پورت: ۲۲
- رمزنگاری شده است
- ابزار ساده برای کپی فایل شبیه دستور cp ولی روی SSH
- برای کپی سریع و یک‌باره خوب است و امکاناتش از SFTP کمتر است

## چه زمانی از کدام استفاده کنیم
- برای بیشتر انتقال‌های امن از SFTP استفاده کن
- برای کپی سریع فایل‌ها از SCP استفاده کن
- FTP را روی اینترنت استفاده نکن مگر داخل یک شبکه خیلی کنترل‌شده
